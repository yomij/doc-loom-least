# Minimal Project Example

This is a tiny repo-shaped example for evaluating Doc Loom Least without
installing skills.

## What To Look At

| Path | Why |
|---|---|
| `AGENTS.md` | Thin project instruction pointer. |
| `docs/product/current-state.md` | Operational input for next-slice discovery. |
| `docs/cases/README.md` | Derived case dashboard. |
| `docs/cases/20260703-demo-fast-path/` | Minimal Fast-Path case with plan and closure. |

## What This Proves

- A project can use the workflow with only Markdown artifacts.
- The dashboard is a derived view, not routing truth.
- The user-facing state is smaller than the internal `case_state.yaml` fields.

## What This Does Not Prove

- It is not an installed Agent Skills runtime.
- It is not a replacement for the full protocol in `skills/_shared/`.
