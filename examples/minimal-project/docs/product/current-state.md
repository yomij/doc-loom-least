Demonstrate a tiny docs-first project that can use Doc Loom Least without a
backend or CLI.

## Target User

## Current Demo State

- A project-level agent entry exists.
- A derived cases dashboard exists.

## Current Bottleneck

protocol.

## Feedback / Signals

## Constraints

- No backend, daemon, scheduler, or workflow runner.
- No automatic execution of recommendations.
